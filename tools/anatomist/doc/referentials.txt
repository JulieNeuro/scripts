Display images in the same referential
======================================

cd /neurospin/brainomics/neuroimaging_ressources

nobias_200.nii.gz: is a typical single subject scan from we have a tranfo to Talairach/AC-PC-Anatomist
T1.nii: is the MNI template (to check if everything is ok)
weights_map_l2.nii: a weigths image (we know that this image contains a tranfo to MNI)


- Run anatomist
anatomist single_subject_mni_icbm200T/default_analysis/nobias_200.nii.gz spm8/templates/T1.nii examples_images/weights_map_l2.nii

- Settings / referential windows
Red: Talairach/AC-PC-Anatomist (anatomist referential)
Purple: MNI

- By default image are shown in Talairach/AC-PC-Anatomist.
For each image (in the anatomist windows) right clic / load information from file header

- Display images, drop into Axial

- Link transformations
  * T1.mni contains a transfo toward MNI
  * weights_map_l2.nii transformation goes toward MNI: In Referentials windows
    create an identity transfo to MNI (purple) left click + CTL
  * nobias_200.nii: load a tranfo from nobias_200.nii to Talairach/AC-PC-Anatomist:
    single_subject_mni_icbm200T/registration/RawT1-200_default_acquisition_TO_Talairach-ACPC.trm
