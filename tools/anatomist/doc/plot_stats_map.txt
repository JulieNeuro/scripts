Palette
=======
Need a palette with transparent zero value.

1 Use predifided palette
------------------------
copy /neurospin/brainomics/ressources/anatomist/palette_signed_values_blackcenter.tiff in ~/.anatomist/rgb/
(create rgb directory if needed)


2 Build custom palette
----------------------

Click right weight image goto palette editor. Design palette using RGBA chanel
     _ 
R __/

G \__/

B _
   \__

=> Check that middle of the palette is black
A ---- (full opacity, eventually add a sharp hole in the middle of the palette)

Usage
-----
Right click on obejct / color / Palette choose "weights_map_blackcenter"
Check value 0 at center

1) Fusion with anat
===================
Objects 
 - weights (A)
 - anat   (B)
Mapping mode: linear / B (anat) if A (weights) is black.

2) Surface rendering
====================

- Mesh larg cluster (> 10) AimsClusterArg => graph
- Create spheres (mesh) for small cluster < 10
  pyaims.SurfaceGenerator
- Load graph => concat meshs + spheres
  SurphaceManip.MeshCat => to mesh


- Fusion: mode Geometrical
- Methods: Point to point with depht offset (inside)
- Submethods: Mean
- Parameters: Deth (mm= 1/2 voxel size typycally 0.75 mm or 0.5 mm
